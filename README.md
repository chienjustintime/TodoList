# TodoList
This is an sample project for demonstrating crud and project layer architectureW

# Technologies #
- ASP.NET MVC5
- Entity Framework 6
- Autofac
- SQL Server Express LocalDB

- DI/IoC

- Repository and Generic Repository
- N-Tier Architecture

This Project is built against ASP.NET MVC5
